# DC MUSIC 🎵
### Premium High-Bass Music Experience

**DC MUSIC** is a high-fidelity, high-performance music player clone based on the Apple Music aesthetic. It is designed for audiophiles who want **extra loudness**, **deep bass**, and access to the world's largest music library via YouTube.

![App Preview](https://images.unsplash.com/photo-1614613535308-eb5fbd3d2c17?q=80&w=1000)

## 🚀 Key Features

- **Unlimited Music Library**: Powered by a robust YouTube Search Engine (via Invidious/Piped bridges). Find any song, artist, or remix instantly.
- **Professional Audio Engine**: Built with the **Web Audio API** for real-time signal processing.
- **Super Bass Booster**: Custom low-shelf EQ allowing up to **25dB of Bass Boost**.
- **Extra Loudness (500% Vol)**: Amplifies sound levels beyond standard browser limits for a powerful listening experience.
- **High-Fidelity UI**: Premium glassmorphism, smooth animations, and dynamic backgrounds that shift colors based on the current album art.
- **Mobile First**: Fully responsive design with a sticky bottom navigation bar and touch-optimized player controls.
- **Privacy Focused**: No tracking, no account required.

## 🛠️ Built With

- **Frontend**: HTML5, Vanilla CSS3 (Custom Properties), Modern JavaScript (ES6+)
- **Audio Logic**: Web Audio API (BiquadFilterNode, GainNode)
- **Engine**: YouTube IFrame API & Invidious Search Proxy
- **Icons**: Lucide Icons
- **Typography**: Apple-style 'Outfit' Google Font

## 🎧 High/Low Volume Buttons

Located in the player bar, you can quickly jump volume levels or use the **Audio Boost Panel** to fine-tune your bass and extra loudness thresholds.
